假设高度已知，请写出三栏布局，其中左栏、右栏宽度各位300px，中间自适应

技术方案比较

方案 优点 缺点
float 兼容性好 浮动元素脱离文档流，需要清除浮动
position 方便快捷 布局脱离文档流，可使用性差
flex 解决上述两种范方案的不足，非常适合移动端布局 兼容性一般
table 兼容性好 当其中某一单元格高度变化时，其他单元格的度也会相应变化
grid 标准化网格布局方案，代码量简化很多 兼容性差

当去掉“高度已知”条件之后

table、flex布局方案依旧坚挺，其余方案已不再满足需求
